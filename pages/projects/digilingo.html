Digilingo: Digital Literacy, Reimagined

Digilingo is a digital literacy application developed in Flutter, aimed at enhancing digital skills in Denver Public Schools (DPS). Below, you can find a project video and key details about the initiative.

Watch the Project Video

Key Features

  • Built using Flutter for cross-platform compatibility.
  • Designed for Denver Public Schools (DPS).
  • Interactive and gamified learning modules.
  • Focus on empowering both students and educators.
